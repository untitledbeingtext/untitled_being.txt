"untitled_being.txt” is an anonymous personal blog I created to share my sudden intertwined thoughts.

Whenever I start naming a file, the system always begins with “Untitled.” Since I see myself as a small nameless being, it naturally became untitled being. I chose the “.txt” extension because it is a versatile format that can turn into image files, HTML, CSS, JS, and more. In the same way, I hope my intertwined thoughts can open up connections between different ideologies and concepts, just as a simple text file holds the potential to become many things.
